Model Selection with Python: An Introduction to Hyper Parameter Tuning

Fri September 04, 02:30 PM–02:55 PM • Back to program
Start time 14:30
End time 14:55
Countdown link Open timer

The number of machine learning algorithms available and the options presented for their configuration presents a bewildering array of choices. This talk will discuss strategies and algorithms for making these choices in a principled way.

The world of machine learning is like a restaurant that presents an incredibly complex menu. There are so many choices for modelling algorithms and settings that it's hard to keep up. This talk discusses strategies for handling these choices in a principled, simple manner.

Patrick Robotham He/Him

Patrick Robotham is Principal Machine Learning Engineer at Nimble.
